# 01. Embedded Systems Fundamentals (Core)

## Overview

This section covers the foundational concepts of embedded systems, including processor architecture, low-level programming techniques, and hardware interfacing.

## Contents

### 1.1 Processor & Architecture
- ARM Cortex-M Architecture
- Register organization and memory mapping
- Interrupt handling and NVIC
- Memory management fundamentals

### 1.2 Low-Level Programming (C/C++ & Assembly)
- Expert-level C programming for embedded systems
- Modern C++ features for embedded applications
- Assembly language and inline assembly
- Build process and toolchain

### 1.3 Hardware Interfacing (Peripherals)
- GPIO operations and configuration
- ADC/DAC and analog interfacing
- Timers and PWM generation
- Communication protocols (UART, SPI, I2C, DMA)

## Learning Objectives

By the end of this section, you should be able to:
- Understand ARM Cortex-M processor architecture
- Write efficient low-level code in C, C++, and Assembly
- Interface with common embedded peripherals
- Implement basic communication protocols
- Manage memory in resource-constrained environments

## Prerequisites

- Basic C programming knowledge
- Understanding of digital logic
- Familiarity with binary and hexadecimal number systems
