# 05. Industrial & IoT Protocols

## Overview

This section covers communication protocols used in industrial automation and IoT applications, from fieldbus systems to wireless connectivity.

## Contents

### 5.1 Industrial Communication
- Modbus protocol (RTU/TCP)
- Profibus and Profinet
- EtherCAT real-time Ethernet
- OPC UA for Industrial IoT

### 5.2 Wireless & IoT
- Bluetooth Low Energy (BLE)
- Wi-Fi for embedded systems
- MQTT and CoAP protocols
- LoRaWAN and Zigbee

## Learning Objectives

By the end of this section, you should be able to:
- Understand industrial communication protocols
- Implement Modbus RTU/TCP communication
- Work with real-time industrial Ethernet protocols
- Integrate wireless connectivity in embedded systems
- Use IoT application layer protocols (MQTT, CoAP)
- Select appropriate protocol for your application

## Prerequisites

- Understanding of networking basics
- Knowledge of serial communication (UART, RS-485)
- Familiarity with TCP/IP stack
- Experience with embedded C programming

## Protocol Selection Guide

**Use Industrial Protocols when:**
- Industrial automation and control systems
- Factory floor equipment communication
- PLC integration required
- Real-time performance critical
- Harsh electromagnetic environments

**Use IoT Protocols when:**
- Internet connectivity required
- Cloud integration needed
- Low power operation essential
- Long-range wireless communication
- Consumer IoT applications
